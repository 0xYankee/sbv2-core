```bash env title="near.env"
CHAIN="near"
# Near Config
NEAR_NETWORK_ID=""
NEAR_RPC_URL=""
NEAR_NAMED_ACCOUNT=""
# Oracle Config
NEAR_ORACLE_KEY=""
HEARTBEAT_INTERVAL=15
# Task Runner Config
TASK_RUNNER_SOLANA_RPC=""
GCP_CONFIG_BUCKET="" # Optional, for private queues
# Keypair Config, one-of required
FS_PAYER_SECRET_PATH=""
DOCKER_PAYER_SECRET=""
GOOGLE_PAYER_SECRET_PATH=""
AMAZON_PAYER_SECRET_PATH=""
# Monitoring Config
METRICS_EXPORTER=""
PAGERDUTY_EVENT_KEY=""
VERBOSE=1
```

## Near Config

<table>
  <thead>
    <tr>
      <th>Env Variable</th>
      <th>Definition</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>CHAIN</td>
      <td>
        <b>
          <u>Required to be set to 'near'</u>
        </b>
        <br />
        <b>Type</b> - aptos / near / solana*
        <br />
        <b>Default</b> - solana
        <br />
        <b>Description</b> - Dictates which chain to listen for on-chain events
        <br />
      </td>
    </tr>
    <tr>
      <td>NEAR_NETWORK_ID</td>
      <td>
        <b>
          <u>Required</u>
        </b>
        <br />
        <b>Type</b> - testnet / betanet / mainnet
        <br />
        <b>Description</b> - Near network ID
      </td>
    </tr>
    <tr>
      <td>NEAR_RPC_URL</td>
      <td>
        <b>
          <u>Required</u>
        </b>
        <br />
        <b>Type</b> - URL
        <br />
        <b>Description</b> - Near RPC URL. Must not be rate limited
      </td>
    </tr>
    <tr>
      <td>NEAR_NAMED_ACCOUNT</td>
      <td>
        <b>
          <u>Required</u>
        </b>
        <br />
        <b>Type</b> - String (Ex. sbv2-authority.near)
        <br />
        <b>Description</b> - On-chain account name of the oracle authority payer
      </td>
    </tr>
  </tbody>
</table>

## Oracle Config

<table>
  <thead>
    <tr>
      <th>Env Variable</th>
      <th>Definition</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>NEAR_ORACLE_KEY</td>
      <td>
        <b>
          <u>Required</u>
        </b>
        <br />
        <b>Type</b> - Address
        <br />
        <b>Description</b> - Base58 address or byte string address of the Near
        oracle account
        <br />
      </td>
    </tr>
    <tr>
      <td>HEARTBEAT_INTERVAL</td>
      <td>
        <b>
          <u>Optional</u>
        </b>
        <br />
        <b>Type</b> - Number (seconds)
        <br />
        <b>Default</b> - 30
        <br />
        <b>Description</b> - Seconds between oracle heartbeats. Queues have
        different oracle heartbeat requirements. Recommended value is 15
      </td>
    </tr>
  </tbody>
</table>
