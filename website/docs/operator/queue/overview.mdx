Anyone is permitted to create their own Switchboard network and run their own
set of oracles.

## Solana

<Tabs>
<TabItem value="javascript" label="Javascript" default>

```js
import { QueueAccount } from "@switchboard-xyz/solana.js";

const [queueAccount, txnSignature] = await QueueAccount.create(program, {
  name: "My Queue",
  metadata: "Top Secret",
  queueSize: 100,
  reward: 0.00001337,
  minStake: 10,
  oracleTimeout: 60,
  slashingEnabled: false,
  unpermissionedFeeds: true,
  unpermissionedVrf: true,
  enableBufferRelayers: false,
});
const queue = await queueAccount.loadData();
```

</TabItem>
<TabItem value="cli" label="CLI">

```bash
sbv2 solana queue create \
    --keypair ~/.config/solana/id.json \
    --size 100 \
    --name "My Private Queue" \
    --reward 0 \
    --minStake 0 \
    --oracleTimeout 300 \
    --slashingEnabled \
    --permissionedFeeds \
    --unpermissionedVrf \
    --enableBufferRelayers \
    --feedProbationPeriod 100 \
    --consecutiveFeedFailureLimit 500 \
    --consecutiveOracleFailureLimit 500
```
