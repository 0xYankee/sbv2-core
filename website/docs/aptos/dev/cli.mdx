# Sbv2 aptos

## Install

```bash npm2yarn
npm install -g @switchboard-xyz/cli^2
```

## Config

```bash
# Testnet
sbv2 config set aptos testnet rpc https://testnet.aptoslabs.com
sbv2 config set aptos testnet default-account "~/.config/.aptos/config.yaml"

# Devnet
sbv2 config set aptos devnet rpc https://devnet.aptoslabs.com
sbv2 config set aptos devnet default-account "~/.config/.aptos/config.yaml"
```
